# 역할 공백 배정 확정 (착수보고서 역할표 보완)

착수보고서(Ⅷ장 역할 분담표) 기준으로 담당자가 명시되지 않았던 항목들을 정리하고 배정을 확정하기 위한 회의 안건.

## 안건

1. 메인 LLM 모델 확정 → 민제 (Haiku/Sonnet 이원화, 사실상 완료 — 회의에서 보고)
2. 사람-LLM 상관도: 평가 "진행"은 민찬 / 통계 "분석·해석"은 민제로 경계 확정
3. 보안/개인정보 대책 → 동훈 백엔드 TODO로 배정
4. UI 법적 고지 문구 → 민찬 첫 원격 태스크로 배정
5. Stretch 항목(외국인 페르소나, OCR, RLHF 탐색)은 미배정 유지, Phase 2 후 재논의
6. API 비용 분담 및 월 지출 한도 합의 ($5 초기 충전, 한도 $10~20)
